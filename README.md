# IP Analyzer

Консольное приложение на C#, которое анализирует список IP-адресов и определяет их географическую принадлежность с помощью публичного API `ipinfo.io`.

Программа загружает IP-адреса в память, получает информацию о стране и городе для каждого IP, группирует данные по странам и выводит статистику в консоль.

---

## Функциональность

- Загрузка списка IP-адресов
- Получение информации по каждому IP через `ipinfo.io`
- Подсчёт количества IP-адресов по странам
- Определение страны с максимальным количеством IP
- Вывод списка городов для этой страны
